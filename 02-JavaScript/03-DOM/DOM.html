DOM - Documnet Object Model

DOM представляет любоую HTML-страницу как сложный объект, который агрегирует в себя другие объекты - элементы страницы. DOM - объединяет в себя все элемены страницы в дерево объектов, корневым элементом которого является объект 'document'. Этот объект всегда неявно создается в любом Java-Script-коде.
